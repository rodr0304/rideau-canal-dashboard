# Rideau Canal Dashboard

## Overview

This project provides a simple web dashboard for the Rideau Canal Monitoring System. It displays sample canal condition data for multiple locations and presents the information in a clean and easy-to-read format.

The dashboard demonstrates how a lightweight backend and frontend can be used to visualize monitoring data.

---

## Features

- Displays monitoring data for multiple Rideau Canal locations
- Shows ice thickness, temperatures, and snow accumulation
- Includes safety status labels
- Uses a simple Node.js and Express backend
- Serves a static frontend with HTML, CSS, and JavaScript

## Purpose

This dashboard is part of the Rideau Canal Monitoring project. It provides a simple interface to visualize monitoring data generated by the sensor simulation and processed through Azure services.
